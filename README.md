# Aplikace Purkyňka
Android aplikace pro správu několika služeb na Střední průmyslové škole elektrotechnické a informačních technologií Brno.

Např.:
 - Správce iSAS
 - Automatické přihlašování k Wi-Fi
 - Správce iCanteen
 - Správce rozvrhů
 - Prohlížeč docházky
